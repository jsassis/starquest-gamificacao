# 🌟 StarQuest — Sistema de Missões

Um sistema de gamificação interativo criado para gerenciar missões, acumular pontos e resgatar recompensas. O projeto roda diretamente no navegador, armazenando o progresso localmente.

## 🚀 Funcionalidades
* **Sistema de Pontuação:** Ganhe pontos ao completar missões diárias ou tarefas específicas.
* **Resgate de Recompensas:** Troque os pontos acumulados por prêmios personalizados.
* **Painel Administrativo:** Área protegida por PIN para gerenciar pontos, editar missões e visualizar o histórico.
* **Armazenamento Local:** O progresso (pontos, histórico, sequências de missões) é salvo automaticamente no navegador (Local Storage), sem precisar de banco de dados.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3 (Estilo Neon e animações)
* JavaScript puro (Vanilla JS para lógica e estado)
